Minicraft: Ultimate Edition

Minicraft was made by @notch for the 22'nd Ludum Dare competition in December 2011.
Minicraft: Ultimate Edition is a mod being developed in @ItsYourPalMike's spare time.

The main purpose of the mod is to allow additional content to be seamlessly added into the game. This is accomplished with a lovely little plugin system. A plugin is a tiny .jar file that contains new mobs, items, tools, weapons, resources, sounds, graphics, menus, world generation code, etc. Pretty cool, huh?

When the game starts you can pick which plugins will be enabled during the session. Official plugins developed by myself or approved authors are available through the in-game downloads menu. Alternatively, plugins can be added by simply dragging the .jar file into your .minicraft folder.
